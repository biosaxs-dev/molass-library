# Draft Response to Reviewer: Python 3.8 Support

Thank you for your question regarding Python 3.8 support.


At present, the package has been tested on Python 3.9. While compatibility with Python 3.8 is expected, it has not been fully verified. Some dependencies may not have been explicitly tested on Python 3.8, and automated tests are not currently run for this version, so compatibility is not automatically checked.

Given that this library is intended for a limited number of expert users, Python 3.8 support will be provided if there is a specific need. If you require Python 3.8 compatibility or encounter any issues, please let us know, and we will address them accordingly.
