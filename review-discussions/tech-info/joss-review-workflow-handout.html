3. Workflow Steps

  1. Reviewer posts a comment
    Status: NEEDS-TECH-INPUT
  2. Technical input provided by Takahashi
    Create file in tech-info/
    Update status: TECH-INFO-PROVIDED
  3. Draft response prepared
    Use response-drafts/
  4. Review and finalize
    Status: READY-FOR-TECH-REVIEW
  5. Bot automates notifications and status updates

4. Status Markers

MarkerMeaning
NEEDS-TECH-INPUTAwaiting technical info
TECH-INFO-PROVIDEDInfo supplied, ready to draft
READY-FOR-TECH-REVIEWDraft ready for review
